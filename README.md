# ATM Management System
# Project Title

ATM Management System

# Project Description

The ATM Management System is a Python-based application that simulates basic ATM operations.
It allows users to perform banking activities like checking balance, depositing money, and withdrawing money securely.

This project helps in understanding:

Conditional statements

Loops

Functions

Basic banking logic

User authentication

# Features

User login with PIN authentication

Check account balance

Deposit money

Withdraw money

Exit option

Displays transaction messages

Prevents withdrawal if insufficient balance

# Technologies Used

Python

VS Code

Command Prompt / Terminal

# How to Run the Project

Open VS Code

Open the project folder

Open Terminal

Run the command:

python atm.py

# Working Process

User enters PIN

If PIN is correct → Access granted

User selects an option:

1. Check Balance

2. Deposit

3️. Withdraw

4️. Exit

System processes the request

Updated balance is displayed
